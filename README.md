# Drip Company

Loja streetwear — preto + azul. Slogan: *"Integre seus criativos"*.

## Como abrir
- **`dripcompany.html`** — versão standalone (todas as imagens embutidas). Abre direto no navegador, sem servidor. Use esse pra mostrar/hospedar rápido.
- **`index.html`** — versão de desenvolvimento (usa imagens em `assets/img/`).

## Recursos
- Catálogo masculino com fotos reais
- Carrinho funcional: adiciona produtos → finaliza pedido direto no **WhatsApp** com a mensagem pronta (produtos, quantidades e total)
- Design responsivo (desktop + mobile no mesmo nível)
- Logo transparente, identidade preto/azul
